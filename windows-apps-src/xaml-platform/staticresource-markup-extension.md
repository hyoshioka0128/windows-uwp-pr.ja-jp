### <a name="an-example-staticresource-usage"></a><span data-ttu-id="232f3-142">{StaticResource} の使用例</span><span class="sxs-lookup"><span data-stu-id="232f3-142">An example {StaticResource} usage</span></span>

<span data-ttu-id="232f3-143">この XAML 例は、[XAML データ バインディング サンプルに関するページ](https://go.microsoft.com/fwlink/p/?linkid=226854)から抜粋したものです。</span><span class="sxs-lookup"><span data-stu-id="232f3-143">This example XAML is taken from the [XAML data binding sample](https://go.microsoft.com/fwlink/p/?linkid=226854).</span></span>

```xml
<StackPanel Margin="5">
    <!-- Add converter as a resource to reference it from a Binding. --> 
    <StackPanel.Resources>
        <local:S2Formatter x:Key="GradeConverter"/>
    </StackPanel.Resources>
    <TextBlock Style="{StaticResource BasicTextStyle}" Text="Percent grade:" Margin="5" />
    <Slider x:Name="sliderValueConverter" Minimum="1" Maximum="100" Value="70" Margin="5"/>
    <TextBlock Style="{StaticResource BasicTextStyle}" Text="Letter grade:" Margin="5"/>
    <TextBox x:Name="tbValueConverterDataBound"
      Text="{Binding ElementName=sliderValueConverter, Path=Value, Mode=OneWay,  
        Converter={StaticResource GradeConverter}}" Margin="5" Width="150"/> 
</StackPanel> 
```

<span data-ttu-id="232f3-144">この例では、カスタム クラスによってサポートされるオブジェクトを、[**ResourceDictionary**](https://docs.microsoft.com/uwp/api/Windows.UI.Xaml.ResourceDictionary) のリソースとして作成しています。</span><span class="sxs-lookup"><span data-stu-id="232f3-144">This particular example creates an object that's backed by a custom class, and creates it as a resource in a [**ResourceDictionary**](https://docs.microsoft.com/uwp/api/Windows.UI.Xaml.ResourceDictionary).</span></span> <span data-ttu-id="232f3-145">有効なリソースにするには、ほかにも、この `local:S2Formatter` 要素に **x:Key** 属性値が必要です。</span><span class="sxs-lookup"><span data-stu-id="232f3-145">To be a valid resource, this `local:S2Formatter` element must also have an **x:Key** attribute value.</span></span> <span data-ttu-id="232f3-146">属性値は "GradeConverter" に設定されます。</span><span class="sxs-lookup"><span data-stu-id="232f3-146">The value of the attribute is set to "GradeConverter".</span></span>

<span data-ttu-id="232f3-147">XAML でもう少し先に進むと、リソースが要求されています。`{StaticResource GradeConverter}` というところです。</span><span class="sxs-lookup"><span data-stu-id="232f3-147">The resource is then requested just a bit further into the XAML, where you see `{StaticResource GradeConverter}`.</span></span>

<span data-ttu-id="232f3-148">{StaticResource} マークアップ拡張を使って、別のマークアップ拡張 [{Binding} マークアップ拡張](binding-markup-extension.md)のプロパティを設定していることに注意してください。つまり、ここではマークアップ拡張が 2 つ、入れ子構造で使われています。</span><span class="sxs-lookup"><span data-stu-id="232f3-148">Note how the {StaticResource} markup extension usage is setting a property of another markup extension [{Binding} markup extension](binding-markup-extension.md), so there's two nested markup extension usages here.</span></span> <span data-ttu-id="232f3-149">内側のものが最初に評価されます。このため、リソースが取得された後、値として使われます。</span><span class="sxs-lookup"><span data-stu-id="232f3-149">The inner one is evaluated first, so that the resource is obtained first and can be used as a value.</span></span> <span data-ttu-id="232f3-150">これと同じ例は、「{Binding} マークアップ拡張」にも記載してあります。</span><span class="sxs-lookup"><span data-stu-id="232f3-150">This same example is also shown in {Binding} markup extension.</span></span>
