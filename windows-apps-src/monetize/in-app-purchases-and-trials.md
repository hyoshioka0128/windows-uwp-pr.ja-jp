<span data-ttu-id="cd79f-105">Windows SDK に用意されている API を使用して、以下の機能を実装し、ユニバーサル Windows プラットフォーム (UWP) アプリによる収益を向上させることができます。</span><span class="sxs-lookup"><span data-stu-id="cd79f-105">The Windows SDK provides APIs you can use to implement the following features to make more money from your Universal Windows Platform (UWP) app:</span></span>
