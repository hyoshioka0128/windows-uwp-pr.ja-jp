<span data-ttu-id="d970c-112">各メソッドは、標準の購入 UI をユーザーに示し、トランザクションが完了すると非同期的に完了します。</span><span class="sxs-lookup"><span data-stu-id="d970c-112">Each method presents a standard purchase UI to the user and then completes asynchronously after the transaction is complete.</span></span> <span data-ttu-id="d970c-113">メソッドは、トランザクションが成功したかどうかを示すオブジェクトを返します。</span><span class="sxs-lookup"><span data-stu-id="d970c-113">The method returns an object that indicates whether the transaction was successful.</span></span>
