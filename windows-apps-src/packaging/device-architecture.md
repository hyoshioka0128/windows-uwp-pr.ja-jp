### <a name="x86"></a><span data-ttu-id="f9ad5-152">x86</span><span class="sxs-lookup"><span data-stu-id="f9ad5-152">x86</span></span>
<span data-ttu-id="f9ad5-153">x86 はほぼすべてのデバイスで実行されるため、一般的に、アプリ パッケージでは x86 を選ぶのが最も安全な構成になります。</span><span class="sxs-lookup"><span data-stu-id="f9ad5-153">Choosing x86 is generally the safest configuration for an app package since it will run on nearly every device.</span></span> <span data-ttu-id="f9ad5-154">Xbox や一部の IoT Core デバイスなど、デバイスの中には x86 構成のアプリ パッケージを実行できないものもあります。</span><span class="sxs-lookup"><span data-stu-id="f9ad5-154">On some devices, an app package with the x86 configuration won't run, such as the Xbox or some IoT Core devices.</span></span> <span data-ttu-id="f9ad5-155">ただし、PC の場合は x86 パッケージが最も安全な選択肢であり、最も広い範囲のデバイスに展開できます。</span><span class="sxs-lookup"><span data-stu-id="f9ad5-155">However, for a PC, an x86 package is the safest choice and has the largest reach for device deployment.</span></span> <span data-ttu-id="f9ad5-156">Windows 10 デバイスの大部分は、引き続き x86 バージョンの Windows を実行しています。</span><span class="sxs-lookup"><span data-stu-id="f9ad5-156">A substantial portion of Windows 10 devices continue to run the x86 version of Windows.</span></span>
