# <a name="style-for-uwp-apps"></a>UWP アプリのスタイル

アプリはスタイルによってさらに引き立ちます。スタイルとは、太字の選択、見た目の選択、控え目な装飾などです。 それはアプリの美的な側面を指します。 

## <a name="style-and-the-fluent-design-system"></a>スタイルと Fluent Design System

<p>Fluent Design System では、ライト、深度、モーション、マテリアル、およびスケールが組み込まれた、モダンでクリーンな UI を作成できます。 ここで説明するスタイル ガイドラインは、色、文字体裁、<a href="../style/acrylic.md">アクリル素材</a>や<a href="../style/reveal.md">表示</a>などの新機能を通じて Fluent Design を作成するために役立ちます。 
</p>
