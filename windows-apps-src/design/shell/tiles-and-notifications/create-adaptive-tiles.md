## <a name="a-basic-example"></a><span data-ttu-id="3a7a1-125">基本的な例</span><span class="sxs-lookup"><span data-stu-id="3a7a1-125">A basic example</span></span>

```xml

      ...
    </binding>
  
    <binding template="TileWide">
```

```csharp
                Children =

                }
            }
        },

    }
```
