# <a name="tiles-badges-and-notifications-for-uwp-apps"></a><span data-ttu-id="9801f-104">UWP アプリのタイル、バッジ、通知</span><span class="sxs-lookup"><span data-stu-id="9801f-104">Tiles, badges, and notifications for UWP apps</span></span>
 

<span data-ttu-id="9801f-105">タイル、バッジ、トースト、通知を使用して、アプリへのエントリ ポイントを提供し、ユーザーに最新情報を提示する方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="9801f-105">Learn how to use tiles, badges, toasts, and notifications to provide entry points into your app and keep users up-to-date.</span></span>

> <span data-ttu-id="9801f-106">**重要な API**: [UWP Community Toolkit Notifications NuGet パッケージ](https://www.nuget.org/packages/Microsoft.Toolkit.Uwp.Notifications/)</span><span class="sxs-lookup"><span data-stu-id="9801f-106">**Important APIs**: [UWP Community Toolkit Notifications nuget package](https://www.nuget.org/packages/Microsoft.Toolkit.Uwp.Notifications/)</span></span>

<p><img style="float: left; margin: 0px 15px 15px 0px;" src="images/tile-and-live-tile.png" />
<span data-ttu-id="9801f-107">タイルとは、スタート メニュー上でアプリを表すものです。</span><span class="sxs-lookup"><span data-stu-id="9801f-107">A tile is an app's representation on the Start menu.</span></span> <span data-ttu-id="9801f-108">すべての UWP アプリにはタイルがあります。</span><span class="sxs-lookup"><span data-stu-id="9801f-108">Every UWP app has a tile.</span></span> <span data-ttu-id="9801f-109">さまざまなタイル サイズ (小、普通、大、ワイド) を有効にすることもできます。</span><span class="sxs-lookup"><span data-stu-id="9801f-109">You can enable different tile sizes (small, medium, wide, and large).</span></span></p>

<p><span data-ttu-id="9801f-110"><em>タイル通知</em>を使用すると、タイルを更新して、新しい情報 (ニュース ヘッドライン、最新の未読メッセージの件名など) をユーザーに伝えることができます。</span><span class="sxs-lookup"><span data-stu-id="9801f-110">You can use a <em>tile notification</em> to update the tile to communicate new information to the user, such as news headlines, or the subject of the most recent unread message.</span></span></p>

<p><span data-ttu-id="9801f-111"><em>バッジ</em>を使用すると、システム提供のグリフまたは 1 ～ 99 の数値の形式で、状態情報または要約情報を提供できます。</span><span class="sxs-lookup"><span data-stu-id="9801f-111">You can use a <em>badge</em> to provide status or summary info in the form of a system-provided glyph or a number from 1-99.</span></span> <span data-ttu-id="9801f-112">バッジは、アプリのタスク バーのアイコンにも表示されます。</span><span class="sxs-lookup"><span data-stu-id="9801f-112">Badges also appear on the task bar icon for an app.</span></span> </p>

<p><span data-ttu-id="9801f-113"><em>トースト通知</em>は、<em>トースト</em> (または<em>バナー</em>) と呼ばれるポップアップ UI 要素を介してアプリがユーザーに送信する通知です。</span><span class="sxs-lookup"><span data-stu-id="9801f-113">A <em>toast notification</em> is a notification that your app sends to the user via a pop-up UI element called a <em>toast</em> (or <em>banner</em>).</span></span> <span data-ttu-id="9801f-114">通知は、ユーザーがアプリを使用中かどうかを問わず表示できます。</span><span class="sxs-lookup"><span data-stu-id="9801f-114">The notification can be seen whether the user is in your app or not.</span></span></p>
<p><span data-ttu-id="9801f-115"><em>プッシュ通知</em>または <em>直接通知</em>は、Windows プッシュ通知サービス (WNS) またはバックグラウンド タスクのいずれかからアプリに送信される通知です。</span><span class="sxs-lookup"><span data-stu-id="9801f-115">A <em>push notification</em> or <em>raw notification</em> is a notification sent to your app either from Windows Push Notification Services (WNS) or from a background task.</span></span> <span data-ttu-id="9801f-116">これらの通知には、関心のあるイベントが発生したことをユーザーに通知することで応答するか、お好みの方法で応答することができます。</span><span class="sxs-lookup"><span data-stu-id="9801f-116">Your app can respond to these notifications either by notifying the user that something of interest happened (via badge update, tile update, or toast) or it can respond in any way of your choice.</span></span></p>
