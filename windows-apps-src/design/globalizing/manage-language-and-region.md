## <a name="app-runtime-language-list"></a><span data-ttu-id="276a5-136">アプリの実行時の言語の一覧</span><span class="sxs-lookup"><span data-stu-id="276a5-136">App runtime language list</span></span>

<span data-ttu-id="276a5-140">具体的には、アプリの実行時の言語の一覧は次の項目で構成されています。</span><span class="sxs-lookup"><span data-stu-id="276a5-140">More specifically, the app runtime language list is made up of these items.</span></span>
