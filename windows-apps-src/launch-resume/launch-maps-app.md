## <a name="launch-a-uri-from-your-app"></a><span data-ttu-id="183c6-123">アプリからの URI の起動</span><span class="sxs-lookup"><span data-stu-id="183c6-123">Launch a URI from your app</span></span>


<span data-ttu-id="183c6-124">アプリから Windows マップ アプリを起動するには、**bingmaps:**、**ms-drive-to:**、または **ms-walk-to:** URI を指定して [**LaunchUriAsync**](https://msdn.microsoft.com/library/windows/apps/hh701476) メソッドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="183c6-124">To launch the Windows Maps app from your app, call the [**LaunchUriAsync**](https://msdn.microsoft.com/library/windows/apps/hh701476) method with a **bingmaps:**, **ms-drive-to:**, or **ms-walk-to:** URI.</span></span> <span data-ttu-id="183c6-125">次の例では、前の例と同じ URI を起動します。</span><span class="sxs-lookup"><span data-stu-id="183c6-125">The following example launches the same URI from the previous example.</span></span> <span data-ttu-id="183c6-126">URI によるアプリの起動について詳しくは、「[URI に応じた既定のアプリの起動](launch-default-app.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="183c6-126">For more info about launching apps via URI, see [Launch the default app for a URI](launch-default-app.md).</span></span>

```cs
// Center on New York City
var uriNewYork = new Uri(@"bingmaps:?cp=40.726966~-74.006076");

// Launch the Windows Maps app
var launcherOptions = new Windows.System.LauncherOptions();
launcherOptions.TargetApplicationPackageFamilyName = "Microsoft.WindowsMaps_8wekyb3d8bbwe";
var success = await Windows.System.Launcher.LaunchUriAsync(uriNewYork, launcherOptions);
```

<span data-ttu-id="183c6-127">この例では、Windows マップ アプリを確実に起動するために [**LauncherOptions**](https://msdn.microsoft.com/library/windows/apps/hh701435) クラスを使っています。</span><span class="sxs-lookup"><span data-stu-id="183c6-127">In this example, the [**LauncherOptions**](https://msdn.microsoft.com/library/windows/apps/hh701435) class is used to help ensure the Windows Maps app is launched.</span></span>
