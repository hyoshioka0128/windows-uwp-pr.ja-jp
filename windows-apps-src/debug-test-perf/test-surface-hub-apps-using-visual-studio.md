## <a name="add-surface-hub-resolutions-to-the-simulator"></a><span data-ttu-id="52825-108">Surface Hub の解像度をシミュレーターに追加する</span><span class="sxs-lookup"><span data-stu-id="52825-108">Add Surface Hub resolutions to the simulator</span></span>

    ```xml
    <?xml version="1.0" encoding="UTF-8"?>
    <ArrayOfHardwareConfiguration xmlns:xsd="http://www.w3.org/2001/XMLSchema"
                                  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">

            <DisplayName>Surface Hub 55"</DisplayName>
    ```

    ```xml
                <Height>2160</Height>
    ```
