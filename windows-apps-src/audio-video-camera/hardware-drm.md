## <a name="override-hardware-drm"></a><span data-ttu-id="85c0d-151">ハードウェア DRM のオーバーライド</span><span class="sxs-lookup"><span data-stu-id="85c0d-151">Override hardware DRM</span></span>

<span data-ttu-id="85c0d-152">このセクションでは、再生するコンテンツがハードウェア DRM (HWDRM) をサポートしていない場合に、ハードウェア DRM をオーバーライドする方法について説明します。</span><span class="sxs-lookup"><span data-stu-id="85c0d-152">This section describes how to override hardware DRM (HWDRM) if the content to be played back does not support hardware DRM.</span></span>

<span data-ttu-id="85c0d-159">次の例は、ハードウェア DRM の除外方法を示しています。</span><span class="sxs-lookup"><span data-stu-id="85c0d-159">The following example shows how to opt-out of hardware DRM.</span></span> <span data-ttu-id="85c0d-160">この操作は、切り替えの前にのみ行う必要があります。</span><span class="sxs-lookup"><span data-stu-id="85c0d-160">You only need to do this before you switch.</span></span> <span data-ttu-id="85c0d-161">また、メモリ内に PlayReady オブジェクトが存在していないことも確認してください。存在する場合は、動作が未定義になります。</span><span class="sxs-lookup"><span data-stu-id="85c0d-161">Also, make sure you don’t have any PlayReady object in memory, otherwise behavior is undefined.</span></span>

```js

localSettings.values["SoftwareOverride"] = 1;
```

<span data-ttu-id="85c0d-162">ハードウェア DRM に切り替えるには、**SoftwareOverride** の値を **0** に設定します。</span><span class="sxs-lookup"><span data-stu-id="85c0d-162">To switch back to hardware DRM, set the **SoftwareOverride** value to **0**.</span></span>

<span data-ttu-id="85c0d-164">ハードウェア DRM であるか、ソフトウェア DRM であるかを確認する最適な方法は、C:\\Users\\&lt;ユーザー名&gt;\\AppData\\Local\\Packages\\&lt;アプリケーション名&gt;\\LocalState\\PlayReady\\\* を調べることです。</span><span class="sxs-lookup"><span data-stu-id="85c0d-164">The best way to tell if you are in hardware DRM or software DRM is to look at C:\\Users\\&lt;username&gt;\\AppData\\Local\\Packages\\&lt;application name&gt;\\LocalState\\PlayReady\\\*</span></span>
