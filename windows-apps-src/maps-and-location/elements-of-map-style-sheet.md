# <a name="map-style-sheet-reference"></a><span data-ttu-id="a5fb9-104">マップ スタイル シート リファレンス</span><span class="sxs-lookup"><span data-stu-id="a5fb9-104">Map style sheet reference</span></span>

<span data-ttu-id="a5fb9-105">JavaScript Object Notation (JSON) を使用して、マップ スタイル シートを作成できます。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-105">You can create map style sheets by using JavaScript Object Notation (JSON).</span></span>

<span data-ttu-id="a5fb9-106">たとえば、次の JSON を使用して、水域を赤で、水域のラベルを緑で、陸地領域を青で表示できます。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-106">For example, you'd use the following JSON to make water areas appear in red, water labels appear in green, and land areas appear in blue:</span></span>

```json
    {"version":"1.*",
        "settings":{"landColor":"#0000FF"},
        "elements":{"water":{"fillColor":"#FF0000", "labelColor":"#00FF00"}}
    }
```
<span data-ttu-id="a5fb9-107">JSON を使用して地図からすべてのラベルとポイントを削除することもできます。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-107">You could also use JSON to remove all labels and points from a map.</span></span>

```json

    {"version":"1.*", "elements":{"mapElement":{"labelVisible":false},"point":{"visible":false}}}
```

<span data-ttu-id="a5fb9-108">最終的な結果を生成するために、プロパティの値が変換される場合があります。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-108">Sometimes the value of a property is transformed to produce the final result.</span></span>  <span data-ttu-id="a5fb9-109">たとえば、表示されるエンティティの種類によって vegetation fillColor の影がやや異なります。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-109">For example, vegetation fillColor has slightly different shades depending on type of the entity being displayed.</span></span>  <span data-ttu-id="a5fb9-110">ignoreTransform プロパティを使用することで、この動作はオフにすることができます。それにより、指定の正確な値が使用されます。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-110">This behavior can be turned off, thereby using the precise provided value, by using the ignoreTransform property.</span></span>

```json
    {"version":"1.*",
        "settings":{"shadedReliefVisible":false},
        "elements":{"vegetation":{"fillColor":{"value":"#999999","ignoreTransform":true}}}
    }
```

<span data-ttu-id="a5fb9-111">このトピックでは、地図の外観をカスタマイズするために使用できる JSON のエントリと[プロパティ](#properties)を示します。</span><span class="sxs-lookup"><span data-stu-id="a5fb9-111">This topic shows the JSON entries and [properties](#properties) that you can use to customize the look and feel of your maps.</span></span>
